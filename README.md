# esp32-breadboard

if you've ever used an esp32 and a breadboard in the same project you would kow the frustration of only being able to use one side of the pins and even so, the side youre able to use only has one pin spot. so i set out to make this 3d model to make it easier to use an esp32 for protoyping. with this you can prototype easily, more efficiently and stylishly.<img width="1920" height="632" alt="breadboard shellsdqq" src="https://github.com/user-attachments/assets/592d4479-dceb-4162-9047-4cb14531adb2" />

# BOM
* 51 metal inserts
*  4 screws
# How to build :
For screws and similar items, it is recommended to find the cheapest in your location. first 3d print the breadboard lid and shell in the CAD folder then manually put the metal inserts in place then screw the lid shut and voila.
